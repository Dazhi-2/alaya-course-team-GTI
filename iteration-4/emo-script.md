# 迭代4 演示脚本

## 教师端
1. 登录 `teacher1` / `teacher123`
2. 进入“我的课程” → 点击“数据结构”的“查看学生名单”
3. 为学生1输入成绩85 → 提示“成绩录入成功”，统计区域更新
4. 修改成绩为90 → 提示“成绩修改成功”，统计更新
5. 演示异常：输入-1 → 前端或后端提示“成绩必须在0-100之间”
6. 演示已退课学生无法录入（如找到 `active=false` 的记录）→ 提示“该学生已退课”

## 学生端
7. 登录 `student1` / `student123`
8. 进入“我的成绩” → 显示已选课程的成绩（已录入显示分数，未录入显示“---”）
